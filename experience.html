College Projects

During my final year at college, I undertook several significant projects that were instrumental in enhancing my technical skills and problem-solving abilities. Below are some key projects I worked on:

1. Campus Event Management System

This web application was designed to streamline the management of campus events. It allowed students to register for events, receive updates, and provide feedback. The system also enabled organizers to manage registrations, send notifications, and analyze participation data.

  • Technologies Used: HTML, CSS, JavaScript, PHP, MySQL
  • Key Learnings: Full-stack development, user authentication, database management, and responsive design.
  • Impact: Improved event participation and communication efficiency on campus.

2. Student Performance Prediction

This project involved creating a machine learning model to predict student performance based on various factors such as attendance, past grades, and participation in extracurricular activities. The goal was to identify students at risk of underperforming and provide timely interventions.

  • Technologies Used: Python, Scikit-learn, Pandas, NumPy
  • Key Learnings: Data preprocessing, model training and evaluation, feature selection, and predictive analytics.
  • Impact: Enabled early identification of at-risk students, allowing for targeted support and improved academic outcomes.

3. Campus Navigation Mobile App

Designed a mobile application to help new students navigate the campus. The app included features like building locations, route planning, and real-time notifications about campus events and closures.

  • Technologies Used: Java, XML, Android Studio, Google Maps API
  • Key Learnings: Mobile app development, UI/UX design, integration of third-party APIs, and testing on multiple devices.
  • Impact: Enhanced the campus experience for new students and visitors by providing easy navigation and timely information.
